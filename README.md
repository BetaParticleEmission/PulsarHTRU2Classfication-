## What is this about? 

The directories contain python files, dataset, and images pertinent to performing classification for pulsars.
Pulsars emit radio waves that pertubate through space which can eventually be detected from researchers
on Earth using large-aperture radio telescopes. The goal of this project is to learn what are the potential factors 
that can lead to classifying radio waves emitted from pulsars rather than RFI (radio frequency interference) and 
other entities. 

## Dataset Source 

To site the publisher of the dataset: 
Lyon, R. (2015). HTRU2 [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C5DK6R.

## How to use 

To make sure the python notebook can run, you must install all of the dependencies. The main 
packages include scikit-learn, pandas, numpy, matplotlib, seaborn, plotly, statsmodels. 

Run the following for proper procedure: 

1. `git clone repo_name`
2. `cd repo_name`
3. `python -m venv .venv`
4. `source .venv/bin/activate.ps1 (powershell) or source .venv/bin/activate (linux)`
5. For Dash: `pip install Dash` and run with `python app.py`
6. For quick dependency install: `pip install -r requirements.txt`
